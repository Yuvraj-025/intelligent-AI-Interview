# VoxHire AI — MVP Scope

## MVP Goal
Build a polished and functional version of the product with the core interview loop working end-to-end.

The MVP should prioritize:
- clean execution
- working adaptive logic
- voice-first interaction
- usable report generation

The MVP should not try to solve everything at once.

---

## MVP Must-Have Features

### 1. User Authentication
- signup
- login
- basic user profile

---

### 2. Interview Start Flow
User can select:
- interview role
- interview mode
- optional difficulty

---

### 3. Voice-Based Question Delivery
The system asks questions through generated voice.

---

### 4. Voice-Based User Answer Capture
The user responds through microphone.

---

### 5. Whisper Transcription
User answer is converted into text.

---

### 6. AI Question Generation
The system generates role-relevant questions.

---

### 7. AI Evaluation
The system evaluates user response quality.

---

### 8. Adaptive Next Question Logic
The next question should depend on:
- previous score
- topic weakness
- mode type

Even simple rule-based adaptation is acceptable in MVP.

---

### 9. Normal Mode
Must support:
- realistic interview pacing
- detailed questions
- at least limited follow-up behavior

---

### 10. Rapid Fire Mode
Must support:
- faster questions
- timer pressure
- quick scoring

---

### 11. Final Report
At the end of a session, the user should receive:
- overall score
- strengths
- weaknesses
- improvement summary

---

## MVP Nice-to-Have (If Time Allows)
- role-specific dashboard
- session history
- transcript review
- downloadable report
- waveform UI
- resume-aware interviewing

---

## Out of Scope for MVP
These should not be built in the first version unless time remains:

- advanced facial emotion detection
- recruiter panel / company portal
- multiplayer mock interview
- live human interview handoff
- company-specific hiring pipelines
- full resume parser automation
- detailed long-term performance analytics

---

## MVP Success Criteria
The MVP is successful if a user can:

1. start an interview
2. hear a question
3. answer by voice
4. get evaluated
5. receive adaptive next questions
6. finish the session
7. view a useful final report
